# dotfiles

These are my dotfiles: `~/.*`

## Prerequisites

* ZSH. To change your default shell to ZSH: `chsh -s $(which zsh) $USER`
* [Homebrew](http://brew.sh/).

## Installation

    $ ./install.sh

## Nice to have

Remap Caps Lock to `esc` with
[Seil](https://pqrs.org/osx/karabiner/seil.html.en).
